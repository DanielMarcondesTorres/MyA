Ook Vamos aprender a utilizar o Mya

  • Primeiro de tudo, baixe os arquivos
  • Você tem duas opções, "MyA.zip" baixar arquivos separados
  • Recomendo a opção do arquivo minificado (.min)
  • Agora você tem dois arquivos, coloque-os na sua pasta
  • agora você tem que adicionar ao código fonte
  • O primeiro (MyAnimate.min.css) Adicione no header como abaixo
  • <link rel="stylesheet" href="MyA/MyAnimate.min.css">
  • O segundo vai antes do término do </html> como abaixo
  • <script type="text/javascript" src="MyA/MyAnimate.js">

Agora vamos aprender como usar as animações

  • Nossa biblioteca é de uso simples, apenas selecione a animação
  • Então clique no botão "Code"
  • Pronto ai você tem todas as classes que precisa
  • Agora adicione tudo no seu elemento dentro do class=""
  • Você pode também ler o arquivo (MyAnimate.css) para entender melhor
